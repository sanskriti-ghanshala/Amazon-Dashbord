# Amazon-Dashbord
Power BI dashboard analyzing Global Superstore sales, profit, customer trends, and business performance.


## 🚀 Project Overview

This project presents an interactive **Power BI Sales Dashboard** built using the Global Superstore dataset. The dashboard provides comprehensive insights into sales performance, profitability, customer behavior, product trends, and regional performance across multiple markets.

The objective of this dashboard is to transform raw business data into meaningful visual insights that support data-driven decision-making.

## 🎯 Business Objectives

- Analyze overall sales and profit performance
- Identify top-performing products and categories
- Monitor regional and market-wise sales trends
- Evaluate customer segments and purchasing patterns
- Discover profitable and loss-making areas
- Support strategic business decisions through visualization
